🔥 Firebase Setup Required

One-time free setup · Takes 3 minutes

1
Go to console.firebase.google.com → Click "Add project" → name it (e.g. vesak-map) → Continue → Disable Google Analytics → Create project
2
In your project → click "Realtime Database" (left sidebar) → "Create database" → choose any location → select "Start in test mode" → Enable
3
Go to ⚙️ Project Settings → scroll down → click "Add app" (Web </>) → register app → copy the firebaseConfig object shown
4
Open this HTML file in a text editor → find the section marked 🔥 PASTE YOUR FIREBASE CONFIG HERE near the top → replace the placeholder values with your copied config → Save → Open in browser
✅ Test mode means anyone with your file can join — perfect for a campus event!
✅ Firebase free tier handles up to 100 simultaneous connections — plenty for a campus.
✅ Share the HTML file with friends — everyone who opens it joins the same live map.
